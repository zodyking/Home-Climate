# Home Climate

Climate monitoring and automation dashboard for Home Assistant with a modern, dark-themed UI.

## Features

- **Zero-config setup** – Add integration, panel appears immediately
- **Room mapping** – Assign climate entities and temp/humidity sensors per room
- **Automation** – Heat/cool thresholds with seasonal logic (outdoor temp or date-based)
- **Presence** – Turn on when person enters zone, off when they leave (configurable durations)
- **TTS** – Customizable announcements on mode changes and presence
- **Admin-only settings** – Full configuration restricted to admin users

## Requirements

- Home Assistant 2024.1.0 or newer
